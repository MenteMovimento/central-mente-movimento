Este manual foi escrito para qualquer pessoa da associação conseguir trabalhar na aplicação sem precisar de saber programação.

1. Acesso

Entrar na aplicação

  1. Abrir o website da aplicação.
  2. Introduzir email e password.
  3. Opcionalmente, ativar a opção para lembrar os dados neste dispositivo.
  4. Clicar em Entrar.

A opção de lembrar dados só deve ser usada em computadores da associação ou dispositivos pessoais protegidos por password.

2. Painel

Painel principal

Pesquisa

Permite procurar por número de sócio, nome, NIF, localidade ou email.

Resumo

Mostra quantos sócios existem e quantos têm quotas em atraso.

Ações

Cada linha permite editar, pagar quota ou eliminar o sócio, conforme o perfil do utilizador.

3. Sócios

Criar ou editar um sócio

Para criar um sócio, clicar em Novo sócio. Para alterar dados existentes, clicar no ícone de lápis na linha do sócio.

Campo Notas de preenchimento
Nº de Sócio Opcional. Se estiver vazio, aparece como Não atribuído.
Nº de Ata de Aprovação Opcional. Se não existir, aparece como número não atribuído.
Data de Adesão Escolher no calendário, quando conhecida.
Última quota paga Selecionar o último ano de quota pago pelo sócio.
CC ou BI Opcional. Recolher apenas quando a associação tiver necessidade real desse dado.
NIF, telemóvel e código postal Devem respeitar o formato esperado para evitar erros na base de dados.
Categoria Campo usado para classificar o sócio conforme a organização da associação.

4. Quotas

Controlar quotas

A aplicação mostra a última quota anual paga. Se o sócio estiver atrasado, a linha fica destacada a vermelho para chamar a atenção.

  1. Confirmar que o pagamento foi recebido.
  2. Clicar em Pagar quota na linha do sócio.
  3. Confirmar a ação para evitar alterações por engano.
  4. A app avança a quota um ano e regista a data do pagamento.

5. Consulta

Pesquisa, filtros e exportação

Filtros rápidos

Todos mostra todos os sócios. Quotas em atraso mostra apenas quem precisa de atenção. Em dia mostra quem tem quotas pagas.

Ordenação

Permite organizar a lista por nome ou outro critério disponível.

Exportar

O menu de três traços permite exportar uma folha de cálculo para backup ou trabalho externo.

Idioma e tema

O mesmo menu permite alternar entre português/inglês e tema claro/escuro.

6. Acessos

Gestor de administradores

O botão Gestor de Administradores aparece apenas a utilizadores com perfil de administrador.

Perfil O que pode fazer
Administrador Gerir sócios, quotas, exportações, histórico e utilizadores autorizados.
Operador Gerir dados de sócios e quotas, sem controlar administradores.
Consulta Consultar informação, sem alterar dados.

Ao criar ou remover administradores, confirmar sempre se a pessoa deve mesmo ter acesso aos dados dos sócios.

7. Auditoria

Histórico de alterações

O histórico ajuda a perceber quem alterou dados e quando a alteração aconteceu.

  • Consultar o histórico quando houver dúvidas sobre uma ficha.
  • Rever alterações importantes após importações, limpezas ou correções em massa.
  • Confirmar se as alterações feitas por novos utilizadores foram corretas.

O histórico não substitui backups. É uma ajuda para rastrear alterações dentro da aplicação.

8. Rotina

Checklist de uso regular

  • Confirmar se novos sócios foram inseridos com nome, NIF, contacto e localidade corretos.
  • Atualizar quotas logo após confirmação do pagamento.
  • Usar o filtro de quotas em atraso para preparar contactos ou relatórios internos.
  • Exportar dados regularmente para ter uma cópia de segurança adicional.
  • Rever a lista de administradores quando alguém entra ou sai da associação.
  • Não partilhar passwords por mensagens abertas.

Problemas comuns

O que fazer quando algo falha

Problema Primeiro passo
Email ou password inválidos Confirmar se a conta existe, se está ativa e se a password está correta.
Não consigo criar ou editar Verificar se o perfil é administrador ou operador.
Dados não aparecem atualizados Atualizar a página com Ctrl + F5 e confirmar se a alteração foi guardada.
Erro ao exportar Tentar novamente, confirmar ligação à internet e informar o responsável técnico.
